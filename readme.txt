== Description ==

This Plugin is uesd to any body who is create a new user to access to wp dashboard. This plugin also create multiple user options and each user get a new user name and password with personal information. 

Cool Features:
*Just put short-code in your page or a Template
*We are trying to validate all of. 
*Multi user creator system. 

== Installation ==

Please follow this Steps:

1. Upload the plugin files to the '/wp-content/plugins/plugin-name' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create a page of name is 'login' and past in it a shortcode of login.
